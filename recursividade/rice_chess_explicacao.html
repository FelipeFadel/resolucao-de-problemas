Problema do Arroz no Tabuleiro de Xadrez (Recursão + Bit Shift)

Ideia do problema

No tabuleiro de xadrez:

  • Casa 1 → 1 grão
  • Casa 2 → 2 grãos
  • Casa 3 → 4 grãos
  • Casa 4 → 8 grãos

Cada casa tem o dobro da anterior.

Precisamos descobrir quantas casas são necessárias para atingir pelo menos um certo número de grãos.

Exemplo:

  • 0 grãos → 0 casas
  • 1 grão → 1 casa
  • 2 grãos → 2 casas
  • 3 grãos → 2 casas
  • 4 grãos → 3 casas

Implementação em TypeScript

class RiceChess {
  public squaresNeeded(grains: number): number {
    if (grains === 0) return 0;
    return this.calculate(grains, 0, 0);
  }

  private calculate(grains: number, square: number, total: number): number {
    if (total >= grains) {
      return square;
    }

    const grainsInSquare = 1 << square;

    return this.calculate(
      grains,
      square + 1,
      total + grainsInSquare
    );
  }
}

Como testar

const rice = new RiceChess();

console.log(rice.squaresNeeded(0)); // 0
console.log(rice.squaresNeeded(1)); // 1
console.log(rice.squaresNeeded(2)); // 2
console.log(rice.squaresNeeded(3)); // 2
console.log(rice.squaresNeeded(4)); // 3

O que é Bit Shift (<<)

O operador bit shift à esquerda desloca os bits de um número para a esquerda.

Exemplo com o número 1 em binário:

1      = 0001
1 << 1 = 0010 → 2
1 << 2 = 0100 → 4
1 << 3 = 1000 → 8

Cada deslocamento multiplica o número por 2.

Ou seja:

1 << n = 2^n

Isso combina perfeitamente com o problema do arroz, porque cada casa do tabuleiro dobra a quantidade de grãos.

Explicação do Código

Classe

Criamos uma classe chamada RiceChess. Isso permite reutilizar e testar facilmente o código.

Método público

squaresNeeded(grains)

Esse método recebe a quantidade de grãos desejada.

  • Se o número de grãos for 0, retornamos 0 casas.
  • Caso contrário chamamos a função recursiva.

Função Recursiva

calculate(grains, square, total)

Parâmetros:

  • grains → quantidade desejada de grãos
  • square → casa atual do tabuleiro
  • total → total acumulado de grãos

Condição de parada

if (total >= grains)

Quando o total acumulado já é suficiente, retornamos o número da casa atual.

Calculando os grãos da casa

const grainsInSquare = 1 << square

Isso gera a sequência:

square 0 → 1
square 1 → 2
square 2 → 4
square 3 → 8

Chamada recursiva

Chamamos novamente a função aumentando a casa e somando os grãos.

calculate(
  grains,
  square + 1,
  total + grainsInSquare
)

Exemplo de execução (grains = 3)

Primeira chamada

square = 0
total = 0

grãos da casa:

1 << 0 = 1

novo total = 1

Segunda chamada

square = 1
total = 1

grãos da casa:

1 << 1 = 2

novo total = 3

Terceira chamada

total >= grains
3 >= 3

Retornamos:

square = 2

Resultado final: 2 casas
